# FNaF 2 Fan Prototype (Night 1)

Небольшая браузерная заготовка фан-игры в духе **Five Nights at Freddy's 2**.

## Что уже есть

- Стартовый экран с кнопкой запуска Ночи 1.
- HUD с временем, зарядом фонарика и статусом музыкальной шкатулки.
- Базовые интеракции: камеры, маска, фонарик, проверка вентиляций.
- Простой игровой цикл для первой ночи (победа в 6:00 AM / поражение при критической угрозе или разряде).

## Как запустить

Можно открыть `index.html` напрямую в браузере или поднять простой сервер:

```bash
python3 -m http.server 8000
```

После этого откройте <http://localhost:8000>.
